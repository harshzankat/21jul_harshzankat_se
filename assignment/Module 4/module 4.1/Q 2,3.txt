1. What is OOP? List OOP concepts
A. - OOP means object-oriented programming.  
   - Object-oriented programming combines a group of data attributes with functions or methods into a unit called an "object."
   - Typically, OOP languages are class-based, which means that a class defines the data attributes and functions as a blueprint for
     creating objects, which are instances of the class.  
   
   ( List of OOP concepts )
	1. Encapsulation
	2. Abstraction
	3. Inheritance
	4. Polymorphism
	5. Class
	6. Object
